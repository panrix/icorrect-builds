# Competitor Benchmarking Blockers

Last updated: 2026-04-01

## Current Blockers

- `Observed`: public competitor data is uneven.
  - some sites show broad menu pricing
  - some show only selected repair examples
  - some reputation data is visible only through Trustpilot/search snippets

- `Unknown`: exact apples-to-apples comparison for:
  - the same model/repair combination
  - parts quality tier
  - warranty terms
  - labour inclusion versus parts-only pricing

- `Unknown`: Google review totals/ratings are not uniformly accessible through a safe public API in the current audit setup.
