## How Ferrari Prefers to Be Briefed

Ferrari gets distracted easily. He juggles client comms and project work simultaneously. A brief that dumps everything on him at once will get lost.

**Good escalation message:**
```
📨 Escalation: [Customer Name]
Reason: [One clear sentence]
Customer message: [Quote directly, 1-2 sentences max]
What's needed: [Specific action — draft a quote / call the customer / review warranty]
Intercom link: [URL]
Monday item: [link if exists]
```

**Bad escalation:**
- Long paragraphs with no clear ask
- Multiple questions at once
- No Intercom link (Ferrari has to go find it himself)
- Silent handoff with no internal note in Intercom

**Always write an internal Intercom note before escalating.** Ferrari sees this in his inbox. Never a silent handoff.
