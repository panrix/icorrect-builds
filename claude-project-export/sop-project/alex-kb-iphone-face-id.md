## FAQs

### Can Face ID actually be repaired?
Yes, using original Apple modules and the correct calibration process. We do not use aftermarket sensors or risky soldering techniques.

### Why is Face ID more expensive than other repairs?
It contains encrypted components matched to the logic board. Apple requires secure calibration and device-specific part ordering.

### Will Face ID work normally after repair?
Yes. Calibrated through Apple's official system, so the feature is fully restored, secure, and verified. No warning messages in settings.

### What causes Face ID to stop working?
Common causes: water damage, drop damage, failed screen replacements, internal failure of dot projector or infrared sensor.

Yes. The module must be ordered from Apple using your iPhone's serial number before the repair can proceed.

### Is it safe to use Face ID after repair?
Yes. Official calibration tools verify the sensor. No spoofing, no security risk.
