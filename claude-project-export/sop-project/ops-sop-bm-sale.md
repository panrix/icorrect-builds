# SOP: BackMarket Sale (Listing and Shipping)

**Status:** Draft v1 (compiled from Ricky Q&A, 2026-04-06)
**Scope:** Post-QC process to get a refurbished device listed on BackMarket and shipped when sold.

---

## 1. Device Passes QC

- QC complete (same process as all repairs, by Ronny)
- Device is already put into packaging post-QC

## 2. Listing

- **Listing script** runs to list the device on BackMarket
- Device waits for sale

## 3. Device Sells

- When sold: a script sends the **shipping label + packing slip** to the Slack general channel
- Previously done by an agent, now a script

## 4. Ship Out

1. Nahid or Ronny picks it up from Slack
2. Insert packing slip
3. Stick shipping label on
4. Tape the package
5. Ship via Royal Mail

---

## Cost Awareness

Every sale incurs:
- 10% BackMarket commission
- Outbound shipping cost
- Original trade-in purchase cost
- Repair/parts cost
- Labour cost

Margin must be tracked per device (ties to profitability calculator in BM Trading SOP).
