# SOP: Mail-In Corporate

**Status:** Draft v1 (compiled from Ricky Q&A, 2026-04-06)
**Scope:** Corporate clients mailing devices for repair.

---

## Flow

Same as [Mail-In Normal Repair](sop-mail-in-repair.md) with the corporate differences:

- **No upfront payment.** Invoiced to the company.
- **Priority repair.** Top of the queue.
- **Confirm return address** - may go to a different office/person than the sender.
- Same corporate pre-checks as walk-in corporate (passcode agreement, IT testing done, who collects/receives back).
