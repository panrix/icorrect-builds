# claude-project-export / sop-project

Pre-existing flat directory of 118 markdown files exported from the upstream Claude project. Categorized by filename prefix. Phase 7a folder-standard rollout left this directory intact (no restructuring of pre-existing subdirs).

## Categories (by filename prefix)

- `alex-*` — Alex CS context, KB pages (per-device repair KB), pricing, glossary, reply templates, triage mapping, customer-lookup, Ferrari guide, Finn lessons, paid-not-contacted SOP, quote SOP, and the alex-kb-INDEX.md
- `bm-sop-*` — Back Market SOP series (00 master + 01-12 covering trade-in, intake, diagnostic, payout, repair-refurb, listing, listings reconciliation, buy-box, sale-detection, shipping, shipment-confirmation, payment reconciliation, Tuesday-cutoff, returns)
- `cs-*` — customer-service: intercom-setup, reply-standards, triage-rules
- `ferrari-writing-library.md` — Ferrari writing-style library
- `finance-*` — finance: payment-truth, revenue-channels, xero-structure
- `foundation-*` — foundation docs: COMPANY, GOALS, PRINCIPLES, PROBLEMS, RICKY, TEAM, VISION
- `kb-*` — KB index and SCHEMA
- `monday-*` — Monday board references: bm-devices, bm-verified-columns, board-relationships, main-board, parts-board
- `ops-*` — operations SOPs: intake-flow, qc-workflow, queue-management, README, BM return, BM return-repair, BM sale, BM trade-in, device-collection, edge-cases-and-verification, mail-in (corporate/diagnostic/repair), quoting-process, walk-in (corporate/diagnostic/simple-repair)
- `pricing-*` — pricing references: ipad, iphone, macbook, services, watch
- `research-*` — research dumps: 11-systems-dump, automation-blueprint, repair-history-analysis
- `team-*` — team: escalation-paths, roster
- `PROJECT-PROMPT.md` — original Claude project system prompt
- `TRACKER.md` — tracker
