# Domain Folders

Use one folder per major operations domain.

Each domain folder should contain:
- `INDEX.md` for control/status
- `process-truth.md` for the main working process document
- `session-notes/` for dated checkpoints
- `references/` for supporting material
- `sops/INDEX.md` for SOP backlog tracking
- `kb/INDEX.md` for KB backlog tracking

Rule:
- do not let `docs/` become a flat pile of domain notes
- domain work should live inside its own folder with an index and output trackers
